Features

Everything you need to run
parallel AI sessions

▦

Multi-session

Spawn as many Claude Code sessions as you need. Each runs in its own PTY with a full terminal emulator. Switch between them instantly.

●

Live status previews

See what every session is doing at a glance — "Reading App.tsx", "Running tests", or "Waiting for input" — without switching to it.

❮❯

Claude Code hooks

Hooks are auto-installed on launch. Airport reads Claude Code's tool activity in real-time so status is always accurate, never heuristic.

Keyboard Shortcuts

Navigate at the speed of thought

Cmd + TNew session
Cmd + WClose session
Cmd + 1 – 9Switch to session 1–9
Cmd + [ / ]Previous / next session
Cmd + JJump to next waiting session
Cmd + KClear terminal

On Linux / Windows, substitute Ctrl for Cmd

Getting Started

Up and running in seconds

1

Install & launch

One command. No clone, no build. npm handles the rest.

npx airport-ai
2

Open sessions

Cmd + T to open new terminals. Run Claude Code in each one.

claude
3

Work in parallel

Status updates appear automatically. Hit Cmd + J to jump to whichever session needs you.

Cmd+J → jump to waiting

Or build from source:

$ git clone https://github.com/tomer-van-cohen/airport && cd airport && npm i --legacy-peer-deps && npm start
